# Facebook Manual Testing Project

## Project Description
This is a manual testing project performed on the Facebook web application,
focusing on Login and Signup modules.

The goal of this project is to validate functional correctness, UI behavior,
input validations, error handling, and performance aspects of the application.

## Modules Covered
- Login Module
- Signup Module

## Types of Testing Performed
- Functional Testing
- UI Testing
- Usability Testing
- Boundary Value Testing
- Negative Testing
- Performance Testing

## Test Artifacts Included
- Test Plan
- Test Cases (Login & Signup)
- Defect Report
- Requirement Traceability Matrix (RTM)
- Test Summary Report

## Test Environment
- OS: Windows 10
- Browser: Google Chrome
- Application Type: Web Application

## Tools Used
- Manual Testing
- Test Case Design Techniques
- Defect Life Cycle

## Key Achievements
- Designed 32+ test cases
- Identified 10 defects (Critical, High & Medium)
- Created complete test documentation lifecycle
