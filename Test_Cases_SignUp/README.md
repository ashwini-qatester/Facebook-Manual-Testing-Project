Project

This project focuses on manual testing of the Facebook Signup functionality.
It validates user registration workflows, field validations, boundary conditions, negative scenarios, UI behavior, and defect identification.

The objective is to ensure that the Signup page works correctly, securely, and provides proper validations for all user inputs.
